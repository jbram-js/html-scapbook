Joe's useful HTML scrapbook

This scrapbook has been put together to list the most useful HTML elements and their usage. Somewhere to reference if ever stuck!

Basic HTML

<!DOCTYPE> - defines the document type (for HTML 5 would be <!DOCTYPE html>)

<html> - defines a HTML document (all your HTML elements sit within this)

<head> - defines information about the document

<title> - defines a title for the document which will be output in the tab through a web browser (sits within the <head>)

<body> - defines the documents body (where main chunk of your html goes!)

Headings

<h1>h1 heading</h1>

<h2>h2 heading</h2>

<h3>h3 heading</h3>

<h4>h4 heading</h4>

<h5>h5 heading</h5>
<h6>h6 heading</h6>

Displaying text

<p> - a paragraph of text

<u> - an underlined paragraph of text

<i> - an italic paragraph of text

<s> - a strikeout paragraph of text
