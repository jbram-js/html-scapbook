Joe's useful HTML scrapbook

This scrapbook has been put together to list the most useful HTML elements and their usage. Somewhere to reference if ever stuck!

Basic HTML

<!DOCTYPE> - defines the document type (for HTML 5 would be <!DOCTYPE html>)

<html> - defines a HTML document (all your HTML elements sit within this)

<head> - defines information about the document. not visible but veryu important for browsers, languages etc

<title> - defines a title for the document which will be output in the tab through a web browser (sits within the <head>)

<body> - defines the documents body (where main chunk of your html goes!). What is seen when you open the page through a browser

<header> - defines the header within a html page

<footer> - defines the footer within a html page

Headings

<h1>h1 heading</h1>

<h2>h2 heading</h2>

<h3>h3 heading</h3>

<h4>h4 heading</h4>

<h5>h5 heading</h5>
<h6>h6 heading</h6>

Displaying text

<p> - a paragraph of text

<u> - an underlined paragraph of text*

<i> - an italic paragraph of text*

<s> - a strikeout paragraph of text*

<b> - a bold paragraph of text*

<q> - signifies a quote

<code> - a piece of computer text

<div> - signifies a section in a document

<span> - also signifies a section in a document

<ul> -

  • an
  • unordered
  • list

<ol> -

  1. an
  2. ordered
  3. list

<br> - single line break in text (does not require </br>)

<hr> - signifies a thematic change in text (does not require </hr>)

Images, video & audio

<img> - adds an image to the page

<img src="burnley.gif" alt="Burnley badge" height="60" width="60" />

Note src - (mandatory) the location of the image relative to location of the HTML file, unless a full (absolute) URL (a.k.a http://) is given

Note alt - (optional) what appears in the tooltip when you hover over the image. Good for accessibility. Best practice.

Note width & height - would be done through css just showing html here

<figure> - good practice when intergrating with css. a figure & caption for image <figure>

<img src="turf-moor.png" alt="Turf Moor" height="120" width="240"> <figcaption>Fig.1 - Turf Moor, Burnley</figcaption> </figure>

<audio> - audio content

<audio controls> <source src="homeshake.mp3" type="audio/mpeg"> </audio>

<video> - video content

Links (anchors)

<a> - a hyperlink (to web page or a page within our site)

<a href="https://www.burnleyfootballclub.com">hyperlink</a> (to web page or a page <a href="linkpage.html">within our site)</a>

Note href - (mandatory) the location of what were are linking to relative to location of the HTML file, unless a full (absolute) URL (a.k.a http://) is given

<link> - link between document & external resource such as stylesheet (will sit in the head of your document)
